Summary
  • This program is a kind of sanction for import and export goods under the project implemented by the parent bank (to hostile countries in the US). with the purpose of restricting import and export) as a function,
  • Export and import documents are OCR (Optical Character Recognition) Inquiry, correct, and check the result of reading the text included in the image by applying AI technology from the image data of the document A program that performs key functions such as examination of import/export sanctions items and approval of examination contents.

    Applied technology
  • By developing the program in the element method (accessory), the program that is actually executed is created as a single JavaScript ( xxx.js file), and it is developed so that it can be directly inserted and applied like a JavaScript component in the necessary project.
  • Any project to which this program (Angular Elements) is applied needs records of input/output, property, and event information of this file. Because it can be directly applied (Embed) as if assembling a Lego block.
  • Angular Material : Basic components (button, input, card, etc.) and table are applied
  • Angular Animation : By applying the Angular Animation function, the table data is expressed vividly.
  • RxJS : All major functions of the program are applied RxJS to improve productivity and maintainability.
  • By applying Angular Cdk, Help Message is expressed in overlay technique. The data is displayed in the desired position naturally without affecting the HTML element in the background.
